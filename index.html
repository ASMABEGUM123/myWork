VideoPlayer with Virtual Buttons
  • =====================================

Here we are upgrading a hotel menu with virtual buttons like Review or Recipe when the user scans the menu with Augmented reality application these buttons can act like a remote to switch between videos. For instance, when the user touches Recipe button the cooking video of corresponding dish will be played in a object plane, placed vertical to the menu card. Similarly the users can also use Review button to see the review video of the hotel.We can customise the menu card according to hotels digital marketing requirements.
